Panda is a simple mod designed (initially) to speed up mass-disenchanting.
This functionality was expanded to include prospecting, gem cutting, milling, glyphs, flasks, elixirs, and quick viewing of AuctioneerAdvanced prices.

To access the panda panel use @/panda@ or the LDB launcher.
